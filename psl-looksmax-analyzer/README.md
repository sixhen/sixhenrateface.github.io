# 👑 PSL Looksmax Analyzer

AI-powered facial attractiveness rating system using ArcFace and MediaPipe.

## 🎯 Features

- **AI-based scoring** using ArcFace embeddings (512D)
- **478 facial landmarks** from MediaPipe
- **23 geometry features** extracted
- **PSL Tier classification** (Sub3 → True Chad)
- **Real-time analysis** via FastAPI
- **Beautiful frontend** with glassmorphism UI

## 🏷️ PSL Tiers

| Score | Tier |

| 8.5-9.3 | 🔥 Chad |
